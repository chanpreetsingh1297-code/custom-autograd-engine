<h3>This folder contains all python scripts for custom autograd engine built from scratch using raw python.</h3>
<h4>File Description:</h4>
<ul>
  <li>__init__py = The main prerequisites for scripts</li>
  <li>engine.py = The main autograd engine python script</li>
  <li>data.py = Synthetic binary classification data generation(for testing purposes)</li>
  <li>nn.py = Script containing individual neurons, layers, and Multi-layer Perceptron(MLP)</li>
  <li>train.py = Training script for model training and testing"</li>
  <li>test.py = Testing script for testing basic engine funcitonality</li>
</ul>
